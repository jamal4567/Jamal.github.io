Maze Project

A 3D Maze Game using Raycasting and SDL2

Play the Game

Key Features

Feature 1: Realistic Maze Environment

The maze is rendered in 3D using raycasting, creating a realistic environment for players to explore and navigate.

Feature 2: Smooth Player Movement

Player movement is handled smoothly, allowing users to walk through the maze with collision detection, adding to the immersive experience.

Feature 3: Dynamic Lighting and Shadows

The game includes dynamic lighting and shadows that change as players move through the maze, enhancing the visual appeal.

About the Project

This project was inspired by my interest in creating immersive gaming environments. It started as a challenge to build a 3D maze using raycasting techniques and SDL2. The development process involved implementing wall rendering, player movement, collision detection, and dynamic camera controls. It's my Portfolio Project for Holberton School.
